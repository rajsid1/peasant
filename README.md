# Peasant

Peasant is ruby library for creating proxies that forwards requests to
another proxies or any nodes.

Peasant is compatible with Tor.

## Installation

You can install it via gem

```bash
gem install peasant
```

Or you can put it in your Gemfile

```ruby
gem 'peasant', '~> 0.0.2'
```
